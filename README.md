

# Ethereum Block Explorer

Ethereum block explorer made with React and web3.js/infura.io

Demo: [https://blockexp.herokuapp.com/](https://blockexp.herokuapp.com/) 

### Installing

```
git clone https://github.com/fabiohild/blockexp
cd blockexp
npm install
npm start
```
